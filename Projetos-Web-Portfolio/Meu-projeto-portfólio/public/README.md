# 💻 Meu Portfólio Web

Este projeto é um **website de portfólio pessoal**, criado para apresentar projetos, habilidades e informações profissionais relacionadas ao desenvolvimento web.

O objetivo é reunir em um único lugar os principais projetos desenvolvidos durante os estudos de programação.

---

# 🎯 Objetivo do Projeto

O site tem como objetivo:

* Apresentar meus projetos
* Demonstrar habilidades em desenvolvimento web
* Servir como portfólio profissional
* Compartilhar minha evolução como desenvolvedor

---

# 🛠 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

---


# 💻 Funcionalidades

* Apresentação pessoal
* Seção de projetos
* Links para redes profissionais
* Layout moderno e responsivo
