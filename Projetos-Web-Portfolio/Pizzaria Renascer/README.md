# 🍕 Pizzaria Renascer — Website

Projeto de desenvolvimento de um website institucional para uma pizzaria, com o objetivo de apresentar o cardápio, informações do estabelecimento e proporcionar uma experiência moderna e intuitiva para os clientes.

Este projeto foi desenvolvido como prática de desenvolvimento web, aplicando conceitos de estruturação de páginas, estilização e organização de código.

---

# 🚀 Objetivo do Projeto

O objetivo deste projeto é criar um site simples, funcional e visualmente agradável para uma pizzaria, permitindo que os clientes possam:

* Conhecer o estabelecimento
* Visualizar o cardápio
* Encontrar informações de contato
* Navegar de forma rápida e intuitiva

Além disso, o projeto serve como prática para aprimorar habilidades em desenvolvimento web.

---

# 🛠 Tecnologias Utilizadas

As principais tecnologias utilizadas no desenvolvimento foram:

* HTML5
* CSS3
* JavaScript
* Node.js (quando aplicável)
* Express.js (quando aplicável)

---


# 💻 Funcionalidades

O site possui as seguintes funcionalidades:

* Página inicial com apresentação da pizzaria
* Seção de cardápio
* Informações sobre o estabelecimento
* Contato e localização
* Layout organizado e responsivo
