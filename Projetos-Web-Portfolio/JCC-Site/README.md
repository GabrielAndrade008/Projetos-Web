# ✝️ JCC — Jovens Caminhando com Cristo

Website desenvolvido para o grupo de jovens **Jovens Caminhando com Cristo (JCC)**, com o objetivo de apresentar o grupo, suas atividades e facilitar a comunicação com os participantes da comunidade.

O projeto foi criado como prática de desenvolvimento web, aplicando conceitos de organização de páginas, estilização e experiência do usuário.

---

# 🎯 Objetivo do Projeto

O objetivo do site é fornecer um espaço digital para o grupo de jovens, permitindo:

* Apresentar o grupo JCC
* Divulgar eventos e encontros
* Compartilhar informações importantes
* Facilitar o acesso às redes sociais e contato

---

# 🛠 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

---


# 💻 Funcionalidades

* Página inicial com apresentação do grupo
* Informações sobre o JCC
* Seção de eventos ou atividades
* Links para redes sociais
* Layout organizado e responsivo

---


# 📈 Melhorias Futuras

* Integração com redes sociais
